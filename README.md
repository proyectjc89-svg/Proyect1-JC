## 📌 Descripción del Proyecto

Este proyecto consiste en una **hoja de vida (CV) en línea**, desarrollada como una página web estática y desplegada en internet mediante GitHub.

El propósito es presentar de forma **clara, moderna y profesional** la información personal, académica y laboral de las autoras, permitiendo que cualquier persona pueda acceder fácilmente desde cualquier dispositivo.

La página incluye secciones organizadas como:

* 👤 Información personal
* 🧠 Perfil profesional
* 💼 Experiencia laboral
* 🎓 Formación académica
* 🛠️ Habilidades
* 📞 Contacto

## 🚀 Tecnologías Utilizadas

El desarrollo se realizó utilizando tecnologías fundamentales del desarrollo web:

* 🧱 **HTML** → Estructura del contenido
* 🎨 **CSS** → Diseño y estilos visuales
* ⚙️ **JavaScript** → Interactividad y dinamismo

---

## ✨ Características Principales

* 🌍 Página web estática (sin backend)
* 📱 Diseño responsive (adaptable a celulares, tablets y computadoras)
* ⚡ Carga rápida y navegación sencilla
* 🔗 Acceso en línea mediante GitHub Pages

---

## 📦 Despliegue

El sitio se encuentra publicado utilizando **GitHub Pages**, lo que permite visualizar la hoja de vida directamente en la web sin necesidad de instalaciones adicionales.

---

## 🎯 Objetivo del Proyecto

Desarrollar una solución digital que permita presentar una hoja de vida de manera **profesional, accesible y moderna**, facilitando su visualización por parte de reclutadores, instituciones y público en general.
